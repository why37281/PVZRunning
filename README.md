# PVZRunning

## 简介

如你所见，这是一个阻止PVZ改版崩溃的程序，但是**不只是PVZ！**

作者对于程序作了一些修改，使其可以适配所有程序

目前有两个版本，都针对Windows x64

1. 只是使用此工具防止PVZ改版崩溃，不想重复输入程序名的用户，建议使用`PVZRunning-special.exe`
2. 需要使用此工具防止其他程序崩溃的，需要使用`PVZRunning-all.exe`
